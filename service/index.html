わいず祭りについて

わいず祭りの名物の一つでもある、屋台通り。
毎年80店舗を超える様々な屋台が一本通りに並びます。

かき氷やリンゴ飴などの甘いお菓子ものや、焼きそばや焼きとりなどの食べ物、さらにはヨーヨーすくいやくじ引きなどの屋台などジャンルも様々で参加いただく方全てが楽しめる空間となっています。

長い一本道にはベンチも設けられているので、休憩がてら屋台の食べ物を口にするのも一興。
ぜひ、お好みの店を見つけてください。

きんぎょすくい / ペットショップでめ

一回 100円

お祭りの王道、きんぎょすくい。ひらひらとプールを泳ぐ金魚は夏の風物詩。 “ポイ”を使って逃げる金魚を上手にすくってみましょう。

くじびき / ぴえろ

一回 100円

たくさんのひもをたぐりよせて、大物商品Getを目指そう。

話題の人気商品や可愛いぬいぐるみ、かっこいい武器など多種多様。

ヨーヨーつり / バブリン

一回 100円

ヨーヨーはもはや映えアイテム。夏の夜を彩るカラフルなヨーヨーは年齢を問わずマッチした夏祭り専用のアクセサリー

糸を垂らして、たくさんつりあげましょう。

やきとり / かける

一本 150円〜

炭火でじっくりと焼き上げた串にかける秘伝のタレをつけてお召し上がりください。

さまざまな種類を用意していますので、お好みの串を見つけてみてください。

やきそば / やまめん

一パック 500円

やまめん独自のやきそばパックを採用し、1パックに溢れんばかりの麺が載せられます。

ソースやのりはお好みでかけていただき、お召し上がりください。

ボールすくい / 清田ゴム

一回 100円

カラフルなスーパーボールが ぐるぐる流れるプールに数多く浮かぶ。

“ポイ”を使ってお気に入りのスーパーボールを手に入れよう。

たこやき / たこやん

一パック 400円

その場作っていつでも熱々のたこ焼きがお手元に。
火傷しない工夫が施されており、熱々なのにぺろりと食べれちゃいます。

チョコバナナ / バナぬ

一本 150円

バナナを覆うチョコレートにカラフルなトッピングをかけてお召し上がりいただけます。

一本だけでも口の中に広がる幸せな甘さできっとご満足いただけるでしょう。

フルーツあめ / のがみ

一本 150円〜

のがみで取り扱っている新鮮なフルーツにサクサクの飴をコーディングしました。

いちごやぶどうなど一口サイズのものから取り扱っています。

焼きとうもろこし / コーンちゃん

一本 200円

畑で取れたプリプリで甘いとうもろこしに醤油を垂らし熱々に仕上げています。

炭の香りも美味しさのトッピング。夏のお供にどうぞ。

りんごあめ / たけうち農園

一本 150円〜

たけうち農園のりんごは密がたっぷりでさわやかな甘さが広がります。

大きな玉から、小さな玉まで、お好みに沿ってお選びください
